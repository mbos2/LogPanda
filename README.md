# Log Panda (IN ONGOING DEVELOPMENT)

Log Panda is a self-hostable audit logging platform.

It provides:

- organization and project management
- API key-based log ingestion
- dashboard authentication with Cognito
- audit log browsing with filtering and pagination
- project-scoped API keys
- self-hostable frontend and backend setup

---

# Features

- multi-organization support
- multi-project support
- organization members and project members
- API key generation per project
- secure audit log ingestion
- audit log querying with filtering and pagination
- Cognito-based authentication
- password reset flow
- email verification flow
- self-hostable architecture
